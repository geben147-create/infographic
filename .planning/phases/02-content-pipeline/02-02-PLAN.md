    Create `scripts/youtube_auth.py`:
    - Takes `--channel-id` and `--client-secrets` CLI args (argparse)
    - Uses `google_auth_oauthlib.flow.InstalledAppFlow.from_client_secrets_file()` with scope `["https://www.googleapis.com/auth/youtube.upload"]`
    - Runs `flow.run_local_server(port=0)` to open browser for consent
    - Saves credentials to `data/yt_token_{channel_id}.json` via `creds.to_json()`
    - Prints confirmation: "Saved YouTube credentials to data/yt_token_{channel_id}.json"
    - If token file already exists, loads it and checks if expired, refreshes if possible

    Create `scripts/download_font.py`:
    - Downloads NotoSansKR-Bold.ttf from Google Fonts GitHub releases

    - Verifies file size > 1MB (Korean fonts are large)
    - Uses httpx for download (already a project dependency)
    - Prints confirmation message
