    - Test: generate_script activity loads channel config, renders Jinja2 template with topic, calls LLM provider, validates Script JSON, saves to run_dir/scripts/script.json

  <action>
    Create `src/services/ollama_client.py`:
    - `class OllamaProvider(LLMProvider)` implementing the ABC from src/models/provider.py
    - Constructor takes `base_url: str` and `model: str` (e.g. "qwen3.5:9b")
    - `async def generate(self, prompt: str, system: str = "", format_schema: dict | None = None) -> str`:
      - Build request body: `{"model": self.model, "prompt": prompt, "system": system, "stream": False}`
      - If format_schema provided, add `"format": format_schema` to body
      - POST to `{self.base_url}/api/generate` via httpx.AsyncClient with timeout=300.0
      - Extract `response.json()["response"]`
      - Strip markdown fences: `text.strip().lstrip("```json").lstrip("```").rstrip("```").strip()`
      - Return cleaned text

    Create `src/activities/script_gen.py`:
    - `class ScriptGenInput(BaseModel)`: topic: str, channel_id: str, run_dir: str
    - `class ScriptGenOutput(BaseModel)`: script: Script, file_path: str
    - `@activity.defn async def generate_script(params: ScriptGenInput) -> ScriptGenOutput`:
      - Load channel config: `config = load_channel_config(params.channel_id)`
      - Parse LLM model spec: `spec = ModelSpec.parse(config.llm_model)`
      - Create provider: `OllamaProvider(base_url=settings.ollama_url, model=spec.model)` for local, raise NotImplementedError for cloud providers (Plan 07 can add)
      - Load Jinja2 template from `settings.prompt_templates_dir / config.prompt_template`
      - Render template with `topic=params.topic, niche=config.niche, tags=config.tags`
      - Define JSON schema for Script model fields (title, description, tags, scenes with narration/image_prompt/duration_seconds)
      - Call `provider.generate(prompt=rendered, system="Respond ONLY with valid JSON.", format_schema=schema)`
      - Parse response: `script = Script.model_validate_json(response)`
      - Save to `{params.run_dir}/scripts/script.json` via `Path(...).write_text(script.model_dump_json(indent=2))`
      - Return `ScriptGenOutput(script=script, file_path=str(script_path))`

    Write `tests/test_script_gen.py`:
    - Mock httpx.AsyncClient.post to return a valid Script JSON
    - Test markdown fence stripping
    - Test template rendering with topic injection
    - Test activity saves script.json to correct path
    - Test invalid JSON raises ValueError with descriptive message
    - Use tmp_path fixture for run_dir
  </action>

        - Return WAV bytes
