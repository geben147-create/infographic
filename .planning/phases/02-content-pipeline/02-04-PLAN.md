    Create `src/services/cost_tracker.py`:
    - `class CostEntry(BaseModel)`: workflow_id: str, channel_id: str, service: str, step: str, amount_usd: float, resolution: str | None = None, timestamp: str (ISO 8601)
    - `class CostTracker`:
      - Constructor takes `log_path: str = "data/cost_log.json"`
      - `def log(self, entry: CostEntry) -> None`: Read existing JSON array, append entry, write back. Use file locking (try `import fcntl` for Unix, fall back to `import msvcrt` for Windows, or use a simple `.lock` file approach). Create file if not exists.
      - `def get_run_total(self, workflow_id: str) -> float`: Read log, sum amount_usd where workflow_id matches.
      - `def get_run_breakdown(self, workflow_id: str) -> list[CostEntry]`: Return all entries for workflow_id.

    Create `src/services/comfyui_client.py`:
    - `class ComfyUIProvider(ImageProvider)`:
      - Constructor: `base_url: str`, `checkpoint: str`
      - `async def generate(self, prompt: str, negative_prompt: str = "", width: int = 1024, height: int = 1024) -> bytes`:
        - Use `asyncio.to_thread()` to wrap sync WebSocket operations
        - Inside thread: connect WebSocket `ws://{host}/ws?clientId={uuid4}`, POST `http://{host}/prompt` with workflow JSON (load from a base template, patch CLIPTextEncode node with prompt text and checkpoint), wait for `executing` event with `node=None`, fetch image via `/view` endpoint
        - Set `ws.settimeout(120)` per Pitfall 2
        - Return image bytes (PNG)

    Create `src/services/fal_client.py`:
    - `class FalImageProvider(ImageProvider)`:
      - Constructor: `model: str` (e.g. "flux-kontext")
      - `async def generate(self, prompt: str, negative_prompt: str = "", width: int = 1024, height: int = 1024) -> bytes`:
        - Call `fal_client.run_async(f"fal-ai/{self.model}", arguments={"prompt": prompt, "image_size": {"width": width, "height": height}})`
        - Download result image URL via httpx
        - Return image bytes

    - `class FalVideoProvider(VideoProvider)`:
      - Constructor: `model: str` (e.g. "kling-2.5-turbo")
      - `async def generate(self, image_path: str, prompt: str, duration_seconds: float = 5.0) -> tuple[str, float]`:
        - Upload local image: `image_url = await fal_client.upload_file_async(image_path)`
        - Call `fal_client.submit_async(f"fal-ai/{self.model}", arguments={"image_url": image_url, "prompt": prompt, "duration": str(int(duration_seconds))})`
        - Iterate events for progress logging
        - Get result, download video to local path
        - Cost lookup: `{"kling-2.5-turbo": 0.07}` per second * duration_seconds
        - Return (video_local_path, cost_usd)

    Create `src/activities/image_gen.py`:
    - `class ImageGenInput(BaseModel)`: scene_index: int, prompt: str, channel_id: str, run_dir: str, negative_prompt: str = ""
    - `class ImageGenOutput(BaseModel)`: file_path: str
    - `@activity.defn async def generate_scene_image(params: ImageGenInput) -> ImageGenOutput`:
      - Load channel config
      - Parse image_model spec
      - If provider is "local": use ComfyUIProvider(base_url=settings.comfyui_url, checkpoint=config.sdxl_checkpoint)
      - If provider is "fal": use FalImageProvider(model=spec.model)
      - Call `image_bytes = await provider.generate(prompt=params.prompt, negative_prompt=params.negative_prompt)`
      - Save to `{params.run_dir}/images/scene_{params.scene_index:02d}.png`
      - Return ImageGenOutput(file_path=...)

    Create `src/activities/video_gen.py`:
    - `class VideoGenInput(BaseModel)`: scene_index: int, channel_id: str, run_dir: str, image_path: str, prompt: str = "", duration_seconds: float = 5.0
    - `class VideoGenOutput(BaseModel)`: file_path: str, cost_usd: float, method: str
    - Helper `def ken_burns_clip(image_path: str, output_path: str, duration_seconds: float = 5.0) -> None`:
      - Use ffmpeg-python: `ffmpeg.input(image_path, loop=1, t=duration_seconds).filter("zoompan", z="min(zoom+0.0015,1.5)", d=int(duration_seconds*25), x="iw/2-(iw/zoom/2)", y="ih/2-(ih/zoom/2)", s="1920x1080").output(output_path, vcodec="h264_nvenc", r=25, pix_fmt="yuv420p", t=duration_seconds).overwrite_output().run(quiet=True)`
      - If NVENC not available (ffmpeg returns error with "encoder" in stderr), retry with `vcodec="libx264"` as fallback
    - `@activity.defn async def generate_scene_video(params: VideoGenInput) -> VideoGenOutput`:
      - Load channel config
      - output_path = `{params.run_dir}/video/scene_{params.scene_index:02d}.mp4`
      - If `config.vgen_enabled and settings.fal_key`:
        - Parse video_model spec
        - Create FalVideoProvider(model=spec.model)
        - `video_path, cost = await provider.generate(image_path=params.image_path, prompt=params.prompt, duration_seconds=params.duration_seconds)`
        - Copy/move downloaded video to output_path
        - Log cost via CostTracker: CostEntry(workflow_id=extract_from_run_dir, channel_id=params.channel_id, service="fal.ai", step=f"video_gen_scene_{params.scene_index:02d}", amount_usd=cost, resolution="480p")
        - Return VideoGenOutput(file_path=output_path, cost_usd=cost, method="ai_video")
      - Else (Ken Burns fallback):
        - Run `ken_burns_clip(params.image_path, output_path, params.duration_seconds)` via asyncio.to_thread
        - Log $0.00 cost entry with service="none"
        - Print to activity log: "Video gen disabled (FAL_KEY not set). Using Ken Burns fallback."
        - Return VideoGenOutput(file_path=output_path, cost_usd=0.0, method="ken_burns")
