    Create `src/activities/video_assembly.py`:
    - `class AssemblyInput(BaseModel)`: scene_count: int, run_dir: str, transition_duration_seconds: float = 0.5
    - `class AssemblyOutput(BaseModel)`: file_path: str, duration_seconds: float, file_size_bytes: int
    - Helper `def build_concat_file(run_dir: str, scene_count: int) -> str`:
      - Create a concat demuxer text file at `{run_dir}/concat_list.txt`
      - For each scene 0..scene_count-1, write: `file 'video/scene_{i:02d}.mp4'`
      - Return path to concat file
    - Helper `def merge_audio_video(video_path: str, audio_path: str, output_path: str) -> None`:
      - Use ffmpeg-python to merge: `ffmpeg.input(video_path)` as video, `ffmpeg.input(audio_path)` as audio, `ffmpeg.output(video, audio, output_path, vcodec="copy", acodec="aac")` 
    - `@activity.defn async def assemble_video(params: AssemblyInput) -> AssemblyOutput`:
      - Verify all scene files exist: `{run_dir}/video/scene_{i:02d}.mp4` and `{run_dir}/audio/scene_{i:02d}.wav` for i in 0..scene_count-1
      - If any missing, raise `ApplicationError(f"Missing scene files for assembly")`
      - Step 1: For each scene, merge scene video + scene audio into `{run_dir}/video/scene_{i:02d}_merged.mp4`
      - Step 2: Build concat file listing all merged MP4s
      - Step 3: Use ffmpeg concat demuxer to join all scenes:
        ```python
        ffmpeg.input(concat_file_path, format="concat", safe=0)
            .output(output_path, vcodec="h264_nvenc", acodec="copy", pix_fmt="yuv420p")
            .overwrite_output()
            .run(quiet=True)
        ```
      - Catch subprocess error — if "h264_nvenc" in stderr, retry with vcodec="libx264", preset="medium", crf=23
      - Output to `{run_dir}/final/final_video.mp4`
      - Calculate file size and duration (from ffprobe or ffmpeg-python probe)
      - Run via `asyncio.to_thread()` since ffmpeg is blocking
      - Return AssemblyOutput(file_path=output_path, duration_seconds=..., file_size_bytes=...)

    Write `tests/test_video_assembly.py`:
    - Mock ffmpeg.run() and subprocess calls
    - Test concat file generation with correct format
    - Test NVENC-to-libx264 fallback logic

    - Test output path is correct
