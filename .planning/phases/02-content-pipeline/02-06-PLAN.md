    - `class UploadOutput(BaseModel)`: video_id: str, youtube_url: str
    - `@activity.defn async def upload_to_youtube(params: UploadInput) -> UploadOutput`:
      - Load channel config to get youtube_credentials_path
      - Load credentials from `google.oauth2.credentials.Credentials.from_authorized_user_file(config.youtube_credentials_path, scopes=["https://www.googleapis.com/auth/youtube.upload"])`
      - If credentials expired, refresh via `creds.refresh(google.auth.transport.requests.Request())`
      - Build youtube service: `build("youtube", "v3", credentials=creds)`
      - Create request body with snippet (title, description, tags, categoryId) and status (privacyStatus)
      - Create MediaFileUpload with resumable=True, chunksize=-1
      - Execute resumable upload via `request.next_chunk()` loop (wrap in asyncio.to_thread)
      - Extract video_id from response
      - Set thumbnail: `youtube.thumbnails().set(videoId=video_id, media_body=MediaFileUpload(thumbnail_path, mimetype="image/jpeg")).execute()`
      - Save updated credentials (refresh token may have changed)
      - Return UploadOutput(video_id=video_id, youtube_url=f"https://www.youtube.com/watch?v={video_id}")

    Create `src/workflows/content_pipeline.py`:
    - `class PipelineParams(BaseModel)`: run_id: str, topic: str, channel_id: str
    - `class PipelineResult(BaseModel)`: video_id: str | None = None, youtube_url: str | None = None, total_cost_usd: float = 0.0, scenes_count: int = 0, status: str = "completed"
    - `@workflow.defn class ContentPipelineWorkflow`:
      - Import all activity I/O models via `with workflow.unsafe.imports_passed_through()`
      - `@workflow.run async def run(self, params: PipelineParams) -> PipelineResult`:
        - Step 1: `setup_pipeline_dirs` on cpu-queue (timeout 30s)
        - Step 2: `generate_script` on gpu-queue (timeout 5min) — script gen uses LLM (GPU via Ollama)
        - Step 3: For each scene in script.scenes:
          - `generate_scene_image` on gpu-queue (timeout 3min)
          - `generate_tts_audio` on gpu-queue (timeout 5min)
        - Step 4: For each scene:
          - `generate_scene_video` on gpu-queue (fal.ai goes via api-queue internally, but activity registered on gpu-queue for Ken Burns) (timeout 10min)
        - Step 5: `generate_thumbnail` on gpu-queue (timeout 3min) — uses ComfyUI or Pillow
        - Step 6: `assemble_video` on cpu-queue (timeout 10min)
        - Step 7: `upload_to_youtube` on api-queue (timeout 15min)
        - Step 8: `cleanup_intermediate_files` on cpu-queue (timeout 30s)
        - Accumulate total_cost from video_gen outputs
        - Return PipelineResult with video_id, youtube_url, total_cost_usd, scenes_count
