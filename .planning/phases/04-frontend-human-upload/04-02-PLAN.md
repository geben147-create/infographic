GET /api/dashboard/runs?limit=20&offset=0&channel_id=optional

```json

    {
      "workflow_id": "pipeline-abc12345",
      "channel_id": "tech_channel",
      "status": "ready_to_upload",
      "total_cost_usd": 1.50,
      "youtube_url": null,
      "started_at": "2026-04-01T10:00:00",
      "completed_at": "2026-04-01T10:15:00",
      "error_message": null,
      "video_path": "data/pipeline/pipeline-abc12345/final/output.mp4",
      "thumbnail_path": "data/pipeline/pipeline-abc12345/thumbnails/thumbnail.jpg"
    }
  ],
  "total": 1,
  "limit": 20,
  "offset": 0
}
```

GET /api/dashboard/costs?days=30&channel_id=optional
Response:
```json
{
  "total_cost_usd": 45.00,
  "days": 30,
  "by_channel": [
    {"channel_id": "tech_channel", "total_cost_usd": 25.00, "run_count": 10},
    {"channel_id": "health_channel", "total_cost_usd": 20.00, "run_count": 8}
  ]
}
```

GET /api/pipeline/{workflow_id}/download — returns video MP4 file
GET /api/pipeline/{workflow_id}/thumbnail — returns thumbnail JPEG file

4. Create `frontend/app.js`:
   - `const API_BASE = window.location.hostname === 'localhost' ? 'http://localhost:8000' : 'http://localhost:8000';`
     (Comment: "Update API_BASE to your FastAPI server URL when deploying to Netlify")
   - State: `let currentOffset = 0; const PAGE_SIZE = 20; let currentChannel = '';`
   - `async function fetchRuns()`:
     - Build URL: `${API_BASE}/api/dashboard/runs?limit=${PAGE_SIZE}&offset=${currentOffset}`
     - If `currentChannel`: append `&channel_id=${currentChannel}`
     - `const res = await fetch(url); const data = await res.json();`
     - Call `renderRuns(data)`
     - Update pagination: `document.getElementById('page-info').textContent = 'Page ' + (Math.floor(currentOffset / PAGE_SIZE) + 1);`
     - Enable/disable prev/next buttons based on offset and total
   - `async function fetchCosts()`:
     - `const res = await fetch('${API_BASE}/api/dashboard/costs?days=30');`
     - `const data = await res.json();`
     - Call `renderCosts(data)`
   - `function renderRuns(data)`:
     - `const tbody = document.getElementById('runs-body'); tbody.innerHTML = '';`
     - For each run in `data.runs`:
       - Create `<tr>` with cells for workflow_id (truncated to 20 chars), channel_id, status badge, cost (formatted `$X.XX` or `--`), started_at (formatted as locale date), completed_at
       - Actions cell:
         - If `run.status === 'ready_to_upload' || run.status === 'completed'`:
           - Download Video button: `<a href="${API_BASE}/api/pipeline/${run.workflow_id}/download" class="btn btn-download">Download Video</a>`
           - Download Thumbnail button: `<a href="${API_BASE}/api/pipeline/${run.workflow_id}/thumbnail" class="btn btn-thumbnail">Thumbnail</a>`
         - Else: empty or "N/A"
     - Populate unique channels into filter `<select>` from data
   - `function renderCosts(data)`:
     - `const container = document.getElementById('cost-cards'); container.innerHTML = '';`
     - Total card: `<div class="cost-card"><div class="amount">$${data.total_cost_usd.toFixed(2)}</div><div class="label">Total (${data.days} days)</div></div>`
     - Per-channel cards: `<div class="cost-card"><div class="amount">$${ch.total_cost_usd.toFixed(2)}</div><div class="label">${ch.channel_id} (${ch.run_count} runs)</div></div>`
   - `function formatDate(isoStr)`: return `new Date(isoStr).toLocaleString()` or `'--'` if null
   - Event listeners:
     - `document.getElementById('refresh-btn').addEventListener('click', refresh)`
     - `document.getElementById('channel-filter').addEventListener('change', (e) => { currentChannel = e.target.value; currentOffset = 0; fetchRuns(); })`
     - `document.getElementById('prev-btn').addEventListener('click', () => { currentOffset = Math.max(0, currentOffset - PAGE_SIZE); fetchRuns(); })`
     - `document.getElementById('next-btn').addEventListener('click', () => { currentOffset += PAGE_SIZE; fetchRuns(); })`
   - `function refresh() { fetchRuns(); fetchCosts(); document.getElementById('last-updated').textContent = 'Last updated: ' + new Date().toLocaleTimeString(); }`
   - Initial load: `refresh();`
   - Auto-refresh: `setInterval(refresh, 30000);`
   - Error handling: wrap fetch calls in try/catch, show "Error loading data" in the respective section if fetch fails
