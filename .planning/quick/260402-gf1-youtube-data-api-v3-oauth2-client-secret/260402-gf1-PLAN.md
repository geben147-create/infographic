<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: GCP Console setup and OAuth2 authorization</name>
  <what-built>Environment verified and ready for GCP Console setup</what-built>
  <action>
The user must complete these manual steps in their browser. Open this URL:

**Step 1: Enable YouTube Data API v3**
URL: https://console.cloud.google.com/apis/library/youtube.googleapis.com

- Select your GCP project (or create one)
- Click "Enable"

**Step 2: Create OAuth2 Desktop credentials**
URL: https://console.cloud.google.com/apis/credentials

- Click "+ CREATE CREDENTIALS" > "OAuth client ID"
- If prompted for consent screen: choose "External", fill minimal info (app name, email), save
- Application type: **Desktop app**
- Name: "YouTube Pipeline" (or any name)
- Click "Create"
- Click "DOWNLOAD JSON" on the confirmation dialog
- Save the downloaded file as: `data/client_secrets.json` in the project directory

**Step 3: Run the auth script**
```powershell
cd C:\WINDOWS\system32
uv run python scripts/youtube_auth.py --channel-id channel_01 --client-secrets data/client_secrets.json
```

This opens a browser window. Sign in with the Google account that owns the YouTube channel. Grant "YouTube upload" permission. The script saves the token to `data/yt_token_channel_01.json`.

**Step 4: Verify the token was created**
```powershell
dir data\yt_token_channel_01.json
```
