PART 04 — Installation

6단계
세팅.

Prerequisites 확인

Node 18+, git, python3. Z님 환경엔 이미 있을 것.

# WSL 터미널에서
node --version  # 18+
git --version
python3 --version

uv 설치 (Spec Kit 의존성)

# astral-sh/uv — 빠른 Python 패키지 매니저
curl -LsSf https://astral.sh/uv/install.sh | sh
source $HOME/.local/bin/env

Spec Kit으로 프로젝트 초기화

# Spec Kit 본체 실행 + Claude Code 모드로 초기화
uvx --from git+https://github.com/github/spec-kit.git \
  specify init my-project --ai claude
cd my-project

이 키트의 md 파일 복사

AGENTS.md, CLAUDE.md, LEARN.md, scratchpad.md, feature 템플릿. install.sh가 자동 처리.

# 이 키트 다운받은 위치에서
bash install.sh my-project

# 또는 수동
cp AGENTS.md CLAUDE.md LEARN.md scratchpad.md my-project/
echo "scratchpad.md" >> my-project/.gitignore

AGENTS.md 프로젝트별 채우기

Commands, Code Style, Prohibitions 섹션을 실제 프로젝트에 맞게 수정.

# 예: Python 프로젝트
[INSTALL_CMD]    →  uv sync
[TEST_CMD]       →  pytest -xvs
[LINT_CMD]       →  ruff check .
[TYPECHECK_CMD]  →  mypy .

Claude Code 시작 → 첫 기능

# Claude Code 실행
claude

# 첫 기능 명세
/speckit.specify "사용자 로그인 기능: 이메일+JWT, rate limit"
/speckit.clarify  # 모호함 Q&A
/speckit.plan     # 아키텍처 + PAL consensus 검증
/speckit.tasks    # DAG 분해
/speckit.implement
