D. Trend Detection (4 Steps)

D-1: keyword_trends + topic_gaps

# Topic Gap Score gap_score = (demand - supply) / demand Where: demand = norm(search_volume) * (1 + growth_rate_30d) supply = norm(video_count) * avg_optimization_strength Output: -1.0 ~ +1.0 Display: 0-100 scale → "Gap: 72 (High Opportunity)"

D-2: keyword_surge (Van Brakel Z-Score)

# Smoothed Z-Score Spike Detection rolling_mean = mean(x[i-lag : i-1]) # lag = 14~30 days rolling_std = std(x[i-lag : i-1]) z = (x[i] - rolling_mean) / rolling_std if |z| > threshold: # threshold = 2.0~3.5 signal = +1 (surge) or -1 (drop) filtered[i] = influence * x[i] + (1 - influence) * filtered[i-1]

D-3: outlier_detect (Modified Z-Score / MAD)

Standard Z-Score X

z = (x - mean) / std

Problem: mean/std 이상치에 왜곡됨. 유튜브 데이터처럼 편향된 분포에서 부정확.

Modified Z-Score (MAD) Best

MAD = median(|xi - median(x)|)

mz = 0.6745 * (xi - median) / MAD

median 기반이라 이상치에 강건. 비대칭 분포에 최적.
