🛠 Architecture

Backend (Python)

  • FastAPI 0.115+ (REST API)
  • Temporal SDK (Workflow orchestration)
  • SQLModel + SQLite (Database)
  • Alembic (Migrations)
  • Pydantic 2.x (Validation)

Frontend (React)

  • React 19 + Vite 8
  • Tailwind CSS 4
  • React Query (Data fetching)
  • Recharts (Charts)
  • React Router (HashRouter)

AI / Generation

  • Qwen3-14B via Ollama (Script)
  • ComfyUI + SDXL (Image)
  • CosyVoice2 / Kokoro (TTS)
  • fal.ai WAN 2.2 (Video)
  • FFmpeg 7.x + NVENC (Assembly)

Infrastructure

  • Docker Compose
  • Temporal Server + UI
  • PostgreSQL (Temporal backend)
  • Redis 7.x (Cache)
  • RTX 4070 8GB (Local GPU)

Worker Pools

  • GPU Worker (max=1) — ComfyUI, Ollama, TTS
  • CPU Worker (max=4) — FFmpeg, file ops
  • API Worker (max=8) — fal.ai, YouTube, Sheets

External APIs

  • fal.ai (Video + Image cloud)
  • YouTube Data API v3 (Upload)
  • Google Sheets API v4 (Data input)
  • Gemini 2.5 Pro (Fact-check)
